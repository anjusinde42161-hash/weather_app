## ğŸ”‘ API Key Used  
This project uses the **OpenWeather API**.

How it works:
1. User enters city  
2. A request is sent to:  
   ```
   https://api.openweathermap.org/data/2.5/weather?q=CITY&appid=YOUR_API_KEY&units=metric
   ```
3. API returns:
   - `temp`
   - `description`
   - `weather icon`
   - `city name`
You must replace:
```js
const apiKey = "YOUR_API_KEY";
```
with your actual key.
